GUIDE STREAMER & OBS

Intégration Pro • Sous-titres • Discrétion
👋 Pour les Créateurs : Ce guide vous explique comment afficher les traductions directement sur votre stream (OBS/Streamlabs) sans capturer l'overlay Windows.

0. Licence & Services Cloud (Nouveau)

1
Activez vos clés dans l'onglet Licence : Clé VTP-... pour le logiciel, puis VCV-... pour débloquer Kommz Voice Cloud.
2
Whisper / XTTS doivent être en ligne : En cas d'erreur de transcription ou de clone, vérifiez les URLs Modal configurées côté serveur.
3
Le mode Hybrid est optionnel mais utile pour le rendu vocal : il améliore le timbre et la couleur de voix. Il est surtout utile si vous voulez un rendu plus naturel sur la sortie vocale, sans changer votre configuration OBS.
Variables Render à vérifier :
MODAL_WHISPER_URL (Whisper), MODAL_XTTS_URL (XTTS) et MODAL_GPTSOVITS_URL (Hybrid distant si activé).

I.bis Mode Hybrid (Rendu Vocal)

1
Activez Hybrid dans l'onglet Intelligence Artificielle : ce mode est recommande si vous voulez un meilleur timbre et une voix plus naturelle en sortie.
2
Renseignez une référence audio propre : utilisez une courte référence vocale propre et exploitable. Sans référence valide, le logiciel retombera sur le flux vocal standard.
3
Langues conseillees : FR, EN, JA, KO et ZH. Pour les autres langues, le logiciel peut revenir automatiquement sur le pipeline vocal classique.
Le mode Hybrid change la qualité de la voix, pas la méthode d'affichage dans OBS. Vos sous-titres continuent de fonctionner via obs_subtitles.txt.
Option OBS alternative (Source Navigateur) :
Si vous préférez un rendu "overlay" sans GDI+, vous pouvez ajouter une Source Navigateur dans OBS avec l’URL : http://127.0.0.1:8770/overlay.html.
Les couleurs MOI / ALLIÉS se règlent dans l’onglet Overlay & Couleurs de Kommz.

I. Activation du Module "Stream Connect"

1
Dans l'interface Kommz (Navigateur) : Allez dans l'onglet Modules Système et activez le bouton Stream Connect.
2
Parlez une fois : Dites une phrase au micro. Cela va créer automatiquement un fichier nommé obs_subtitles.txt dans le dossier du logiciel.

II. Intégration OBS Studio (Overlay Pro)

Suivez ces étapes pour avoir des sous-titres dynamiques qui ne bougent pas.

1. Ajouter la Source

  • Dans OBS, panneau Sources, cliquez sur +.
  • Choisissez Texte (GDI+).
  • Nommez-la "Sous-titres IA".

2. Configuration (Critique)

⚠️
RÉGLAGE IMPORTANT :
Cochez la case ☑️ Lire à partir d'un fichier (Read from file).
Cliquez sur "Parcourir" et sélectionnez le fichier obs_subtitles.txt.

3. Le Secret du Centrage Parfait (Anti-Décalage)

Pour éviter que le texte ne se décale quand la phrase change de longueur :

Tout en bas des propriétés ☑️ Utiliser des extensions de texte personnalisées
Largeur (Width) 1920 (Taille écran)
Alignement Centré (Center)
Retour à la ligne Word Wrap

III. Nouveautés Studio & Sécurité

🛡️ Privacy Sentinel (TUTO)

Ce module coupe le son si vous leakez une info privée.

Comment gérer la liste (Zone Rouge) :
  • Bouton 👁️ (Cacher/Voir) : Cliquez dessus pour flouter vos mots secrets si vous montrez l'écran en live.
  • Ajouter un mot : Tapez votre adresse ou nom de famille et faites "Ajouter".
  • Détection Auto : Les numéros de téléphone (06...) sont bloqués automatiquement sans rien faire.

🌍 Polyglot (2ème Langue)

OBS

Pour afficher une traduction (ex: Espagnol) en plus du Français.

  • Activez le module Polyglot.
  • Dans OBS, créez une DEUXIÈME source texte.
  • Sélectionnez le fichier : obs_subtitles_ES.txt.

Nouveau V4.9 : Pack Expressivite

Vous pouvez maintenant piloter le rendu vocal dans l'onglet IA/Hybrid avec 4 reglages utiles pour le live :

  • Sons expressifs : active ou lisse rire, hesitation, soupir, souffle et interjections.
  • Profil : Gaming (plus vivant), Pro (plus propre), Roleplay (plus colore).
  • Transcript : garder les sons dans le texte ou les ignorer pour un overlay plus clean.
  • Rendu TTS : style pour le timbre, neutre pour une voix plus sobre.

IV. Modules Essentiels pour Streamer

🥷 Stealth Mode

Cache la fenêtre noire (CMD) de la barre des tâches. Indispensable pour ne pas leaker votre IP en Alt-Tab.

🛡️ Tilt Shield

Remplace les insultes par des mots gentils. Idéal pour éviter les bannissements Twitch dus aux textes toxiques à l'écran.

🎙️ Tactical Macros

Commandes vocales :
• "Clip ça" -> Lance un replay (Alt+F10)
• "Mute Discord" -> Coupe le son (Ctrl+Maj+M)

📌 Smart Marker

Dites "Marqueur" pour noter un timecode. Un petit "Bip" confirme l'action. Fichier CSV généré à la fin.
