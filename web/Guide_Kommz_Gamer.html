I. Configuration Windows (Critique)

Pour éviter l'écho, le son de l'IA ne doit pas revenir dans votre micro.

Installation VB-CABLE (Windows) - Étape par étape

1
Téléchargez VB-CABLE
Récupérez le lien inclus dans l'email d'accès, ou téléchargez le pack officiel VB-CABLE Driver sur le site officiel VB-Audio.
2
Fermez les applications audio
Fermez Discord, jeux, OBS, navigateur et toute app qui utilise le son.
3
Installez en administrateur
Lancez VBCABLE_Setup_x64.exe (ou x86) en mode Exécuter en tant qu'administrateur, puis cliquez sur Install Driver.
4
Redémarrez Windows
Redémarrage recommandé pour charger correctement les périphériques.
5
Vérifiez les 2 périphériques
Dans Windows > Son, vous devez voir :
  • CABLE Input (VB-Audio Virtual Cable) en lecture
  • CABLE Output (VB-Audio Virtual Cable) en enregistrement
6
Désactivez "Écouter ce périphérique"
Ouvrez les propriétés de CABLE Output, onglet Écouter, puis décochez Écouter ce périphérique.
Conseil : gardez votre vrai casque et votre vrai micro en périphériques par défaut Windows. Le routage vers VB-CABLE se fait ensuite dans Kommz et dans le jeu.
🎧 LECTURE (Playback)
Ce que vous entendez

Gardez votre Casque Principal par défaut.

🎙️ ENREGISTREMENT
Ce que vous dites

Gardez votre Vrai Micro par défaut.

⚠️ IMPORTANT : Dans les propriétés de "CABLE Output", onglet "Écouter", DÉCOCHEZ "Écouter ce périphérique".

Configuration dans Kommz

1
Sortie TTS (Output) : Sélectionnez CABLE Input (VB-Audio).
L'IA va "parler" dans ce câble virtuel pour que le jeu l'entende.
2
Entrée Écoute (Input) :
👉 Pour JOUER : Sélectionnez Votre Micro Principal.
C'est le réglage par défaut pour parler en jeu et sur Discord.
👉 Pour les VIDÉOS/STREAMING : Sélectionnez Mixage Stéréo.
Cela permet à l'IA d'entendre le son de votre PC (YouTube, Netflix...) au lieu de votre voix.

Configuration Discord (Impératif)

⚠️ FILTRES À DÉSACTIVER
Discord tente de "nettoyer" le son, ce qui peut couper la voix de l'IA. Allez dans Voix & Vidéo et désactivez :
  • Suppression du bruit (Krisp / Standard) -> METTRE SUR "AUCUN"
  • Réduction de l'écho
  • Contrôle automatique de gain

IV. Streaming & Mobile

🔴 Mode Streaming Flow

Idéal pour regarder des vidéos YouTube, Twitch ou des films en langue étrangère.

Comment l'utiliser
1. Dans Kommz > Entrée (Input), sélectionnez "Mixage Stéréo" (ou le micro qui capte le son PC).
2. Lancez votre vidéo.
3. L'IA traduira en continu (toutes les 4 secondes) sans attendre les silences.

📱 Mobile Bridge (Web App)

Scannez le QR Code au lancement pour transformer votre téléphone en télécommande. Cela permet de changer la langue cible ou de couper le micro sans faire Alt-Tab en plein jeu.

V. Overlay & Raccourcis

L'overlay est une fenêtre "Fantôme" (Click-through) qui reste toujours au-dessus du jeu.

OBS (optionnel)
Vous pouvez aussi afficher les sous-titres dans OBS via une Source Navigateur : http://127.0.0.1:8770/overlay.html
Les couleurs MOI / ALLIÉS se règlent dans l’onglet Overlay & Couleurs.

⌨️ COMMANDES

Touche F3 (Monitoring Audio) : NOUVEAU

Active/Désactive le retour son IA dans votre casque.

● ON : Vous entendez l'IA.
● OFF : Silence pour vous (Alliés entendent).
Touche F4 (Mode Bypass) :

Coupe l'IA. Votre micro passe en direct (Bip Aigu).

Touche F8 (Panic Reset) : CRITIQUE

Réinitialise immédiatement tout le système audio et les flux bloqués.

🎨 COULEURS

Configurable via l'interface Web.

MOI : Couleur de vos traductions (Cyan).
ALLIÉS : Couleur des joueurs détectés (Jaune).
SYSTEME : Messages d'état (Violet).

VI. FAQ & Dépannage

⚡ PERFORMANCES & SÉCURITÉ
🔐 Ma clé ne s'active pas / erreur serveur
Vérifiez d'abord les points suivants :
  • Email licence identique à celui utilisé à l'achat
  • Clé VTP/VCV copiée sans espace
  • Connexion internet active (activation côté serveur)
Si l'erreur persiste, transmettez log_discret.txt au support.
🚀 Est-ce que Kommz fait baisser mes FPS
Non, absolument pas.
Kommz utilise une architecture déportée (Cloud Groq LPU). Le calcul de l'IA ne se fait PAS sur votre processeur. L'impact sur votre PC est équivalent à celui d'une page web ouverte (0-1% CPU).
🚀 Pourquoi la traduction est-elle si rapide (300ms)
Moteur Deepgram Nova-2.
Contrairement aux autres logiciels, Kommz n'attend pas que la phrase soit finie. Le moteur "Turbo" découpe et traduit l'audio toutes les 0.3 secondes.
🛡️ Risque de Ban (Vanguard, Ricochet)
Risque Zéro.
Kommz fonctionne comme une application externe (comme Discord ou Spotify). Il ne touche jamais à la mémoire du jeu et n'injecte aucun code. L'overlay est une fenêtre transparente standard de Windows.
🔴 AUDIO & STREAMING
🔇 Je n'entends pas l'IA, mais mes alliés oui
C'est le comportement par défaut (Anti-Echo).
Si vous voulez entendre la traduction : Appuyez sur la touche F3 (Monitoring ON).
🔊 Je parle doucement / Je chuchote, ça marche
OUI (Auto-Gain).
L'algorithme détecte les voix faibles et les amplifie numériquement avant de les envoyer à la traduction.
📱 MOBILE & CONNEXION
📲 Le QR Code reste sur "Chargement..."
Vérifiez que votre réseau Windows est sur "Privé" et non "Public". Le pare-feu bloque parfois la connexion locale. Rafraîchissez la page (F5).
🔧 OUTIL DE DIAGNOSTIC (SAV)

🥷 STEALTH MODE (Logs Automatiques)

Ce mode s'active automatiquement au premier lancement du logiciel pour détecter les erreurs.

En cas de problème, envoyez simplement le fichier log_discret.txt généré dans le dossier au support technique.
